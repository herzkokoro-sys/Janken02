# じゃんけんゲーム ✊✌️✋

## 1. 制作アプリのタイトル

じゃんけんゲーム

## 2. 制作アプリの説明

jQuery を使って作成した、ブラウザで遊べるじゃんけんゲームです。
難易度（かんたん / ふつう / むずかしい）を選び、グー・チョキ・パーの
いずれかをクリックするとコンピュータと勝負できます。
コンピュータの手は ✊✌️✋ が高速で切り替わり、難易度によって
切り替えスピードが変化します（速いほど読み取りにくく、難しくなります）。
勝敗が決まるとボタンが押せなくなり、「もう一回」でリセットして再戦できます。

## 3. 工夫した点

- コンピュータの手を `setInterval` で高速に切り替え、じゃんけんの
  ワクワク感（手が回っている演出）を表現した。
- アニメの中身を `cycleHand()` 関数に切り出し、難易度ボタンで
  速度（600 / 400 / 80 ミリ秒）を変えられるようにした。
- 勝敗が決まったら `lockHands()` で手・難易度ボタンをすべて無効化し、
  決着後の連打を防止した。
- あいこのときはアニメを再開させ、そのまま勝負を続けられるようにした。
- 勝者には「Winner!! 🎉」、敗者は手を半透明にするなど、結果を
  見た目でわかりやすくした。

## 4. 難しかった点

- 勝敗が表示されると `result` の手が左にずれてしまい、レイアウトを
  崩さずに表示する方法を見つけるのに苦労した。
- 勝敗確定後にボタンを押せなくする処理を、どこにどう書けばよいか
  （関数化してまとめる方法）を理解するのに時間がかかった。
